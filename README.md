# ATM Fraud Awareness Simulator 💳

An interactive web-based simulator that teaches users how to safely use ATMs and avoid fraud.

## 🚀 Features

* ATM simulation (withdraw, balance, PIN setup)
* Fraud awareness scenarios
* Step-by-step guided tasks
* Interactive UI

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript

## ▶️ How to Run

1. Clone the repo
2. Open `index.html` in browser
